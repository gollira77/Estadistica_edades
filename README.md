# Trabajo Práctico - Estadística con Edades

## Descripción
Este proyecto realiza un análisis estadístico sobre un conjunto de edades de estudiantes, incluyendo medidas de tendencia central, dispersión y percentiles.

## Conceptos aplicados
- Media, mediana y moda
- Varianza y desviación estándar
- Rango
- Percentiles (Q1, Q2, Q3)
- Visualización de datos

## Tecnologías
- Python
- NumPy
- Pandas
- Matplotlib
